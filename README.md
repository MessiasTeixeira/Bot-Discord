# Dice Discord Bot

Bot de rolagem de dados para o Discord, utilizando apenas JS e API do discord

## Status
🚧 Em desenvolvimento — funcionalidades e estrutura podem (irão) mudar (em algum momento).

## Funcionalidades atuais
- [X] Rolagem básica de dados (ex: !roll d20)
- [X] Rolagem com modificador (ex: !roll d20+5)
- [X] Múltiplos dados (ex: 2d6)
- [X] Comandos adicionais básicos, como Clear, Userinfo 
- [ ] Histórico de rolagens
